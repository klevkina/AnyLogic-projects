# AnyLogic-projects
Коллекция имитационных моделей для анализа сложных систем и процессов

## Metro Project
Имитационная модель станции метрополитена для комплексной оценки пассажиропотока, выявления узких мест и оптимизации работы транспортного узла.
Модель позволяет анализировать:
- Загрузку станций и составов в пиковые/непиковые часы
- Время ожидания пассажиров на платформах
- Эффективность различных интервалов движения
- "Бутылочные горлышки" в пассажирских потоках

# Samokat Project
Имитационная модель сервиса доставки "Самокат" для проведения экспериментов по нагрузке на систему и поиску оптимальных операционных параметров.
Возможности модели:
- Тестирование различных сценариев нагрузки
- Определение оптимального количества курьеров
- Анализ эффективности распределения заказов
- Оценка времени доставки и уровня сервиса
